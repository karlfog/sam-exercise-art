# figures-v2 — unified stick-figure set

Original CC0 stick-figure exercise art for the TrainerSam daily email.
ONE style across all 21 program moves: identical proportions, stroke weight,
and brand-red accents (bands, motion arrows, sequence numbers). Each PNG is a
2–3-state composite (start → mid → end) so the movement is unambiguous.
Rasterized from hand-authored SVG (Gmail renders PNG, not SVG). Public domain.
