# sam-exercise-art

Original line-art exercise illustrations (stick-figure cues) used by the
TrainerSam daily coaching email. Hand-authored SVG, rasterized to PNG so they
render inline in Gmail (which does not display SVG).

These original figures are released under **CC0 1.0** (public domain).
The email also hotlinks separate Everkinetic illustrations, which are
CC-BY-SA 4.0 and attributed in the email footer.
