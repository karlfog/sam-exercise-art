# sam-exercise-art

Line-art exercise illustrations (movement cues) for the TrainerSam daily
coaching email, which hotlinks them as PNGs via commit-pinned
raw.githubusercontent.com URLs (Gmail renders PNG, not SVG).

Three generations live here:

- **`figures-v3/`** — the current production set (34 figures). Line-art
  renders **generated with Gemini** by trainerSam's
  `scripts/illustrate_batch.py`, anchored to a shared line-art style
  reference. Not hand-authored.
- **`figures-v2/`** — 56 program moves in one unified stick-figure style,
  hand-authored SVG rasterized to PNG. Being superseded move-by-move as v3
  renders land.
- **Root `*.png`** — the original 8-figure v1 set (hand-authored SVG rasters).
  Retired; only historical email logs pinned to old commits still reference
  them.

Everything here is released under **CC0 1.0** (public domain dedication; note
the figures-v3 images are AI-generated). The email also hotlinks separate
Everkinetic illustrations, which are CC-BY-SA 4.0 and attributed in the email
footer.
